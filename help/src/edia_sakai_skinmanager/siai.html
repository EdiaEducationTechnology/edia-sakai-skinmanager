Skin Manager: Admin Info

This section contains all the information a sakai administrator needs to know.

Permissions

The Edia Skin Manager has the following permissions:

skinmanager.view
view the available skins in the system. This also includes the details.
skinmanager.create
create and upload a new skin. The tool does not alow existing skins to be overwritten this way.
skinmanager.edit
delete an installed skin, if not in use. The system does not allow the deletion of skins that are in use.
skinmanager.delete
change a existing skin by upload. This includes the overwriting of existing skins.
