Skin Manager: New Skin

Upload a new skin

On the index page, if the user has the right permissions, the option "new" is available. Selecting that option will provide the option to install a new skin. This dialog requires two fields:

  1. Name, the name must not be in use, replacing skins can be done by the update in the skin details dialog.
  2. File, the skin zip file must meet the requirements of a valid skin zip file. Please see: the skin zip file documentation for details.
